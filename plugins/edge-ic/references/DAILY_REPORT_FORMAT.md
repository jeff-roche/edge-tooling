# Daily Report Format

Format for daily standup reports posted to Slack.

## Emoji Format

- `:done-circle-check:` - Completed items
- `:in-progress:` - In-progress items
- `:jira-blocker:` - Blocked items

## Structure

```text
Daily Report:

:done-circle-check: Accomplishment 1
:done-circle-check: OCPEDGE-123: Accomplishment with ticket (https://redhat.atlassian.net/browse/OCPEDGE-123)
:in-progress: Work in progress
```

## Important

- Include header line for `/copy` command alignment
- Use plain URLs (Slack will linkify automatically)
- Be concise but complete
- Consolidate similar items
